# Lawteck Peças — Landing Page

Landing page para venda de placas eletrônicas de ar-condicionado (novas e seminovas).

## Stack

- React 19 + TypeScript
- Vite 6
- TailwindCSS (CDN)
- Lucide React (ícones)

## Desenvolvimento

```bash
npm install
npm run dev       # Dev server na porta 3000
npm run build     # Build de produção
npm run preview   # Preview do build
```
